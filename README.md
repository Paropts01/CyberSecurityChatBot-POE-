# CYBERSECURITY AWARENESS CHATBOT

# Overview
The CyberSecurity Awareness Chatbot is a C#-based console application
designed to teach users about cybersecurity practices through engaging in
easy to follow conversations. It covers topics like creating strong passwords,
recognizing phishing attempts, and browsing the web safely, teaching users to
make smarter and good choices in their online activities.

# Features
- Personalised Interaction = The chatbot asks for the user’s name and uses it throughout
  the conversation to create a more engaging experience.
- Professional Conversational Style = Responses are designed to be clear, informative,
  and engaging while maintaining a professional tone.
- Cybersecurity Education Covers key topics including, Password safety, Phishing attacks, Safe browsing practices.
- Typing Animation Effect = Simulates real-time conversation by displaying responses
  with a typing effect.
- ASCII Visual Interface = Displays a styled ASCII logo and structured console layout
  for better user experience.
- Input Validation = Handles empty or invalid inputs gracefully and prompts the user to
  re-enter their question.
- Exception Handling = Uses try-catch blocks to prevent crashes and ensure smooth 
  program execution.
- Welcome Audio = Uses a recording that is converted to wave soud so it can be supported in the help.

# How It Works
When the application starts, the logo will appear follwed by the welcome audion,
then the user is greeted with a console style interface.
They are then asked to enter their name. The chatbot then begins an interactive session where
the user can ask questions related to cybersecurity.
The chatbot processes user input using keyword detection and provides detailed, easy 
to understand responses. If the input is not recognised, the chatbot guides the user 
toward supported topics.
